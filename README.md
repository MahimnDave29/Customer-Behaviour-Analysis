# Customer Behaviour Analysis

Analyzed customer purchasing patterns, subscription behavior, revenue distribution, and demographic insights to support data-driven retail decision-making.

# Dataset
https://www.kaggle.com/datasets/sahilislam007/shopping-trends-and-customer-behaviour-dataset

# Project Overview

Total Customers: 3.9K

Average Purchase Amount: $59.76

Average Review Rating: 3.75

Subscription Split: 27% subscribed | 73% non-subscribed

# Analysis 
Revenue & Sales Analysis

- Revenue breakdown by Category (Clothing, Accessories, Footwear, Outerwear)
- Sales volume comparison across product categories
- Revenue and sales distribution by Age Group

Customer Segmentation

- Age Groups: Young Adult, Adult, Middle-aged, Senior
- Gender-based filtering
- Subscription status analysis
- Shipping type preferences

# Business Value

- Identifies high-revenue customer segments
- Highlights subscription penetration opportunities
- Supports category-level sales optimization
- Enables targeted marketing strategies
